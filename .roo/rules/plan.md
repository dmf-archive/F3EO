# 未来实验计划

## Larger LR Testing: 三阶方法对大学习率的鲁棒性验证

当 `η` (学习率) 过大时，标准优化器会因 `η · λ_max > 1` (其中 `λ_max` 是 Hessian 最大特征值) 而发散。F3EPI 的 `δ_meta` 项对 `||g||` (梯度范数) 的剧烈变化敏感，能在参数进入高曲率盆地前提供反向推力，从而维持 `η · λ_eff < 1` 的有效条件。

因此，F3EPI 的三阶项 `δ_meta` 可作为动态阻尼器，通过主动避开高曲率区域，使优化器能在其他方法会崩溃的超大学习率下保持稳定。

**实验设计** (判别性实验):

- **对照组**: AdamW, AdaHessian
- **实验组**: F3EPI
- **变量**: 学习率 `η`，例如 `1e-3` (基线), `5e-3`, `1e-2`。
- **任务**: CIFAR-10 (快速验证)
- **成功标准**: 在 `η = 5e-3` 或 `1e-2` 时，对照组出现 `NaN` 或性能崩溃，而 F3EPI 仍能收敛到有效解。

**状态**: 待当前实验完成后。

## F3E-Turbo: 混合二阶与三阶的加速与塑造统一

将 F3EPI 的“几何塑造”能力与二阶方法的“收敛加速”能力结合，构建一个混合阶优化器。验证是否能同时获得二阶方法的快速收敛和三阶方法的泛化性能。

**更新规则**:
`Δθ = -η · H⁻¹(g + β · δ_meta)`

- `H⁻¹`: 二阶曲率预处理器，提供“最速收敛”方向。
- `g + β · δ_meta`: F3EPI 的“有效梯度”，由一阶准确性项和三阶几何正则化项构成，提供“最优盆地”方向。

**实验设计**:

- **对照组**: AdamW, AdaHessian, F3EPI (纯三阶)
- **实验组**: F3E-Turbo (F3EPI + AdaHessian 预处理)
- **任务**: CIFAR-10, WikiText-2
- **指标**: 收敛步数、最终精度、遗忘率 (CL)、训练稳定性

**状态**: 待 F3EPI baseline 套件完成后实现。

## F3EPI-ZPD: 预测完整性引导的最近发展区优化

F3EPI-ZPD 把被动梯度下降升维成可微分课程设计：用实时预测完整性 PI 作为模型认知坐标，动态软化交叉熵损失 `L = w(PI)·CE`。高 PI 时 w→1，全力学习；低 PI 时 w→0，抑制梯度，实现维果茨基“最近发展区”的可计算脚手架。

此时“教师”由外部真理 y_true 与内部信念 y_pred 动态合成，PI 权重在每一步仲裁应更多服从世界还是保留偏见，使巩固与探索保持自适应平衡。样本若落在 ZPD 内则全力更新，在外则策略性等待，防止灾难性冲击。

该变体待 F3EPI baseline 套件完成后实现，用于持续学习场景。
